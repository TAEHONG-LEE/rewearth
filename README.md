## 앱 구성

### 사용자 앱 (User App)
일반 사용자를 위한 의류 대여 서비스 앱
- 회원가입/로그인
- 홈 화면 (배너, 카테고리, 추천 상품)
- 카테고리별 상품 조회
- 상품 검색
- 상품 상세 정보 및 대여 예약
- 마이페이지

### 관리자 앱 (Admin App)
서비스 운영자를 위한 관리 앱
- 대시보드
- 상품 관리
- 주문/대여 관리
- 회원 관리
- 정산 관리
